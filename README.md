# rail-shooter
Mini 3D action-based rail-shooter experience

Controls:
W - Up / 
S - Down / 
A - Left / 
D - Right / 
LMB - Fire lasers /  
CTRL + TAB - Tab out of fullscreen 

Install instructions: 
1. Download and extract zip 
2. Open Builds folder 
3. Run RailShooter

Scripts for game can be found in Scripts folder
